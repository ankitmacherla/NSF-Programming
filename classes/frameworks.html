Frameworks

A framework is used for when developers want a certain feature or set of features in their site. Instead of writing their own code, they make use of other peoples' code. Some frameworks include:

  • React is MIT's JavaScript library for UI (user interface). One example of a site is the WhatsApp site.
  • NodeJS is MIT's JavaScript library for executing server-side code. It helps create dynamic webpages. An example of a site that uses NodeJS is PayPal.
  • Bootstrap is an independent CSS library for UI. One site that uses Bootstrap is the Apple site.
