Tips

  1. Save your work!
  2. Use proper syntax. The compiler reads the code EXACTLY as you write it, so be very careful what you instruct it to do.
  3. Comment your code. It helps you later, when you have to update it or give it to someone else.
  4. If you're confused, look it up! Google, StackOverflow, and the language's documentation are great places to start.


    5. Code not working for some reason?
    1. Find out where it's not working. Is it the first half of your code or the second half? Which lines? Sometimes adding 'print statements' (statements that tell the compiler to print at a certain spot) can help locate an error.
    2. Go through the code, line by line, asking yourself two things. 1: What is this line supposed to do? What did I tell it to do? 2: Is this proper syntax?
    3. Use a debugger (not for HTML/CSS). Most compilers have debuggers, which helps you go through your code line-by-line and view the value of every variable at each line. This helps you see exactly what the compiler is doing.
