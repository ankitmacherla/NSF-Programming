HTML

HTML stands for Hypertext Markup Language. It is used to tell the web browser what to display and how to display it.

We always start HTML files with the tag <!DOCTYPE html>. Each webpage has a <head> tag and a <body> tag. Below is an image that explains the function of <head> and <body>.

Image from Zoomin

The <div> tag is used when we want to split our webpage into multiple different pieces. Moreover, we can add up to two types of identifiers to every tag:

  1. id: a unique identifier for the tag.
    • Each element can only have one id
    • Each page has only one element with that id
  2. class: a non-unique identifier for the tag.
    • An element can have multiple tags
    • Multiple classes can have the same element

Yes, a tag can have one id and multiple classes!

Some specific tags we talked about are:

CSS CODE SPECIFYING STYLING
</style>

<a> Used to add a link. <a href="LINK OR FILEPATH">display text </a>
<br> Creates a new line. <p> firstline <br> secondline <p>
