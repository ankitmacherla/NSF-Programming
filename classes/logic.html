Logic

A boolean can have one of two values: true and false.
When writing if, while, or for loops, we need something to tell when to stop the loop. This is in the form of a true or false.

For example, if you wrote the following if statement:

if(true){
//print "Hello, World"!
}
The code would display "Hello, World!" every single time the if statement was checked. Similarly, if we wrote:
if(false){
//print "Goodbye!"
}

"Goodbye" would never be printed.

When comparing two things, there are four categories we generally put the comparison in:

  1. Equal (==). To check if a variable x equals a variable y, we write: x == y.
  2. Not equal (!=). To check if a variable x does not equal variable y, we write: x!=y.
  3. Greater than and less than (> and < respectively).
  4. Greater than or equal to. To check if x is greater than or equal to y, we write: x >= y.
  5. Less than or equal to. To check if a is less than or equal to b, we write a <= b.

What if we want to check for more than one thing? What if I want to check if a number is greater than 6 and also not equal to 9? What if I'm checking if a number is greater than 6 or less than 2?
We can check for many different things:

  • Check 1 AND check 2: To check for AND, use &&. So, to check if x is greater than 6 and not equal to 9, I write x > 6 && x != 9.
  • Check 1 OR check 2: To check for OR, use ||. To check if y is greater than 6 or less than 2, I write y > 6 || y < 2.
