Math

In math, we have four main operations: addition, subtraction, multiplication, and division. With code, there are five:

  1. Addition. We use "+" in code, like we do in math.
    Note: This extends beyond just numbers. You can name two variables add them together, or you can add strings like so: "Hello" + " " + "World!" = "Hello World!". Addition is the ONLY mathematical operation that can be done on strings.
  2. Subtraction. We use "-".
  3. Multiplication. "*".
  4. Division. "/".
  5. Remainder or Modulus ("%"). This operation checks the remainder when the first number is divided by the second. So, 24 % 5 = 4 and 25 % 5 = 0. This is especially helpful for checking odd and even numbers (n % 2 == 0 or n % 2 == 1).
If you want to say, for variable x, that we want to add n to x, we can write: x += n instead of x = x + n. We can also use:
  • x -= n for subtraction
  • x *= n for multiplication
  • x /= n for division
  • x %= n for remainder/modulus

We can also use x++ or x-- for x += 1 and x -= 1.
