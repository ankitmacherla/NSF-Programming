Projects

What now? Here are some ideas:

  • Learn Python or Java
  • Create an update a website/blog for youself, your family, or your class.
  • Make a simple game using HTML, CSS, and JS.
  • Make a calculator app on thunkable!
  • Make a site that quizzes you on your vocabulary.
  • Write a program that, given any number, tells you the following things:
    • Its prime factorization
    • The sum of the primes
    • The sum of its digits
    • All the prime numbers less than the given number
  • Try some of the Project Euler problems.
  • Write a "guess-the-number" game, in which a player guesses a randomly chosen number that the computer chooses. Make sure you check the number that the user inputs and tell the user if it's too high or too low.
  • (JS, Python, Java, etc.) - store information into a text file! Create an address book.
