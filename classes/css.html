CSS

CSS stands for Cascading Stylesheet. It's used to style your webpage, instead of using <font> or <style>. It is especially useful when your website has multiple pages. For example, every page on this website is linked to this CSS file. So, instead of writing many lines to style every page identically, only one line is repeated: the line that links the current file to the .css file.

A CSS file is linked directly to a HTML file
